# User Persona Generator

### About
This projected was created for the 2026 She Innovates Hackathon at the University of Pittsburgh. It seeks to address the PNC Challenge to design an AI-powered productivity solution for Project Managers. To assist product managers in developing personas for development or market execution, this application takes in user input about a product and returns three possible user personas.


### Technologies
This project was built using the Flutter framework and integrates Ollama AI using the Flask framework for Python.
